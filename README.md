A powerful, interactive question sheet organizer designed to help you track your coding practice progress. Built with modern web technologies for a smooth and responsive experience.

## ‚ú® Unique Features

*   **Hierarchy Management:** Organize your practice into **Topics**, **Sub-topics**, and **Questions**.
*   **Drag & Drop Everything:** Seamlessly reorder topics, move sub-topics between topics, or drag questions across different categories with specific animations for a premium feel.
*   **Smart Progress Tracking:** Visual progress bars and completion counters help you stay motivated.
*   **"Focus" Mode:** Collapsible topics and sub-topics allow you to focus only on what you are currently working on.
*   **Interactive Notes:** take markdown-style notes for every question to remember your approach.
*   **Random Question Picker:** Stuck on what to solve? Let the app pick a random uncompleted question for you.
*   **Starred Items:** Mark important questions or topics to revisit later.
*   **Dark Mode UI:** A sleek, modern interface built with **Shadcn UI** and **Tailwind CSS**.

## üì¶ Installation & Setup

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/coder-writes/question-sheet-codolio.git
    cd question-sheet-codolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser.

4.  **Build for production:**
    ```bash
    npm run build
    ```
